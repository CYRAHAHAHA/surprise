## Required File

| Filename               | Description                                 |
| ---------------------- | ------------------------------------------- |
| `background-music.mp3` | Main background music that plays throughout |

## Tips

- Use a soft, romantic instrumental track
- Longer tracks (3-5 minutes) that loop well work best
- Keep file size reasonable (under 10MB for faster loading)
- Supported formats: `.mp3`, `.wav`, `.ogg`

## Ideas for Music

- Soft piano instrumentals
- Acoustic guitar love songs (instrumental)
- Your "our song" instrumental version
- Lo-fi romantic beats

## Volume

The default volume is set to 30% in the config.
You can adjust it in `src/data/config.ts` under `backgroundMusic.volume`.
